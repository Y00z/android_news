# 新闻客户端

>这是自己的第二个安卓项目，代码逻辑非常简单明了，涉及到大部分的安卓知识点，适合初识安卓的朋友学习。


>此app需要数据源，解压源代码中的zhbj.rar压缩包到手机根目录
>然后开启http服务并且是8080端口。
>这里推荐一个安卓http服务程序

>链接: http://pan.baidu.com/s/1ge2gfM7 密码: 98uc

>把`/stirage/emulated/0/`设置根目录，并且web端口8080即可
